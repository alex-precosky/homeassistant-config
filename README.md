# Home Assistant Configuration

This is my configuration repo for [Home
Assistant](https://www.home-assistant.io/). My Home Assistant is hosted in a
docker container managed by [Portainer](https://www.portainer.io/) running in
[Microsoft Windows Subsystem for Linux 2 / WSL2](https://learn.microsoft.com/en-us/windows/wsl/)
on a Windows 11 desktop.
